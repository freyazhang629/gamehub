About Run 3

Run 3 is an incredibly addictive endless running game where you race through dangerous space tunnels! Navigate through a vast galaxy of holes and obstacles as you run through 3D tunnels. The goal is simple: don't fall into space!

Choose from multiple characters with unique abilities and explore hundreds of levels. The game features both Explore Mode and Infinite Mode. In Explore Mode, you'll discover new areas and complete challenging levels. In Infinite Mode, see how far you can run in the endless tunnel!

How to Play

  • Arrow Keys / WASD: Move and jump
  • Spacebar: Jump over gaps and obstacles
  • Wall Running: Run on walls and ceilings to avoid falling
  • Character Selection: Choose different characters with unique abilities
